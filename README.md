# # NP Runtime Studio v1.0 Preview

## Overview

NP Runtime Studio is a lightweight bridge between Visual Studio 2012 and modern AI tools.

Instead of creating temporary files, the selected code is stored as a runtime context and shared with a local Chrome Extension.

## Features

* Send selected code from Visual Studio 2012
* Display project and file information
* View selected code inside Chrome
* Copy code to clipboard
* Save selected code as a file
* Modular architecture for future AI tools

## Getting Started

1. Run NP.Host.
2. Enable the Visual Studio Add-in.
3. Install the Chrome Extension.
4. Select code inside Visual Studio.
5. Right-click and choose **Send To Chrome Bridge**.
6. Open the Chrome Extension and press **Receive**.

## معرفی

NP Runtime Studio یک ابزار کمکی برای Visual Studio 2012 است که امکان ارسال کد انتخاب‌شده به مرورگر و استفاده از ابزارهای مبتنی بر هوش مصنوعی را فراهم می‌کند.

در این نسخه فایل موقت ساخته نمی‌شود و اطلاعات کد به صورت Context در حافظه Runtime نگهداری می‌شود.

## امکانات

* ارسال کد انتخاب‌شده از VS2012
* نمایش اطلاعات پروژه و فایل در Chrome Extension
* نمایش کد انتخاب‌شده
* کپی سریع کد
* ذخیره کد به فایل
* معماری قابل توسعه برای ابزارهای AI

## نحوه استفاده

1. برنامه NP.Host را اجرا کنید.
2. افزونه Visual Studio را فعال کنید.
3. Chrome Extension را نصب نمایید.
4. روی کد موردنظر در Visual Studio راست‌کلیک کنید.
5. گزینه Send To Chrome Bridge را انتخاب کنید.
6. در پنجره افزونه Chrome روی Receive کلیک کنید.
